# Whatsapp Phone Numbers Scraper

- Target : `https://web.whatsapp.com/`

## Requirements

- Chrome Webdriver
- Python >= 3.5

## Setup

```
git clone https://knowbee@bitbucket.org/fastaio/whatsapp_phone_numbers.git
```

```
cd whatsapp_phone_numbers
```

```
python -m venv .
```

```
activate.bat
```

```
pip install -r requirements.txt
```

## Usage

```
  python run.py
```
